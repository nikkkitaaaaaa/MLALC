# MLALC
Modular Linear Algebra Library in C

## Features
- Matrix utilities
- Matrix operations
- LU decomposition
- QR decomposition
- Least squares solver
- Real dataset demonstration

## Build

make

## Run tests

./test_utils
./test_ops
./test_luqr

## Example: Dataset Solver

./solver datasets/beijing_clean.txt

## Applications
- Scientific computing
- Numerical simulations
- Embedded numerical systems
- Environmental data analysis
